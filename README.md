## most-simple-mcd

most-simple-mcd是一个minecraft开服辅助工具，只需鼠标“点点点”，即可开服。搭配自带网页管理后台，使开服以及管理服务器更简单！

## 关于MCDaemon-go
MCDaemon-go是most-simple-mcd的前身，在MCDaemon-go中踩的坑和不合理规划，都将在most-simple-mcd中被填平！
目前MCDaemon-go已经停止维护。
